9. Markdown高级技巧

9.1 支持HTML

Markdown 语法支持 HTML 标签，因此，我们可以插入 HTML 代码。
目前支持的 HTML 元素有：<kbd> <b> <i> <em> <sup> <sub> <br>等

例子:Ctrl+Alt+Del 重启电脑

9.2 转义字符

Markdown 使用反斜杠 \ 来转义特殊字符，如果你需要在文本中插入反斜杠，那么你需要使用两个反斜杠 \\ 。

9.3 高亮代码

Markdown 语法支持代码高亮，只需要在代码块的第一行或第二行指定一种语言，如： ```C++ 。

目前支持的语言有：c/c++ java python javascript html css ruby swift php sql xml yaml json bash markdown

9.4 目录

9.4.1 自动生成目录

Markdown 语法支持目录，只需要在文档的最开始加上 [TOC] 标签，然后 Markdown 软件就会自动生成目录。

9.4.2 手动生成目录

9.5 任务列表

Markdown 语法支持任务列表，只需要在列表项前加上 [ ] 或 [x] 符号，然后 Markdown 软件就会自动生成任务列表。

例子：

  • 任务1
  • 任务2
  • 任务3

9.6 跳转

Markdown 语法支持跳转，只需要在文档中插入锚点，然后在其他地方插入链接，Markdown 软件就会自动跳转到锚点处。
